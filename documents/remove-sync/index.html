removeSync(path)

Removes a file or directory. The directory can have contents. Like rm -rf.

  • path <String>

Example:

1const fs = require('fs-extra')
2
3// remove file
4fs.removeSync('/tmp/myfile')
5
6fs.removeSync('/home/jprichardson') // I just deleted my entire HOME directory.
